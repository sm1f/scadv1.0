Parts are basic building parts.

These should use <use ...scad>.
Includes would get things that are for debugging.

Naming convention

var starting with caps are vectors usually 3 (x, y, z)

prefixes:
cZ: added color of sub parts
tZ: translated simple z
rZ: rotated z
trZ:  translated rotated z

myTypes:
  Block
  NotchedBar
